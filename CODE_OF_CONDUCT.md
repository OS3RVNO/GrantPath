# Code of Conduct

This project adopts the Contributor Covenant Code of Conduct.

## Our Pledge

We want GrantPath to be a respectful, constructive and technically honest project. Contributors, maintainers and users are expected to treat each other professionally and with empathy.

## Our Standards

Examples of behavior that contributes to a positive environment:

- giving clear and respectful technical feedback
- being honest about limits, bugs and tradeoffs
- accepting review comments without hostility
- documenting assumptions and risks
- helping others reproduce issues and validate fixes

Examples of unacceptable behavior:

- harassment, insults or personal attacks
- bad-faith arguments or deliberate obstruction
- publishing private information without permission
- hostile or dismissive responses to bug reports or beginner questions

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing this code of conduct.

## Scope

This code of conduct applies within project spaces and in public spaces when someone is representing the project.

## Enforcement

If you experience or witness unacceptable behavior, report it through the contact path in [SECURITY.md](./SECURITY.md) or through the maintainers listed on the repository.
